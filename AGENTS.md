This project contains sample code for six patterns implementing AI workflows with Lambda durable executions.

Each pattern is implemented in Python (`/python`) and Typescript (`/typescript`)

More information is in the @README.md file
More information on Durable Functions https://docs.aws.amazon.com/lambda/latest/dg/durable-functions.html
Typescript SDK https://github.com/aws/aws-durable-execution-sdk-js
Python SDK https://github.com/aws/aws-durable-execution-sdk-python

Information on contributing to the Github repository is in @CONTRIBUTING.md
